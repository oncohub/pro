How to Use

Browser Requirement

  • Chrome 70 or higher
  • Safari 12 or higher
  • Microsoft Edge 17 or higher (IE not supported)
  • Firefox 63 or higher

Install

iOS (iPhone)
  1. Scan the barcode above or go to the website https://oncohub.github.io/pro/.
  2. Tap the Share button.
  3. From the options, tap the 'Add to Homescreen' option, then you can notice an icon of 'CTCAE 5.0' added to your devices homescreen instantly.
  4. Tap the icon from homescreen, then 'CTCAE 5.0' will be loaded.
Android
  1. Scan the barcode above or go to the website https://oncohub.github.io/pro/.
  2. You can see a Add to Home screen prompt from the bottom of screen. If not, tap the menu button.
  3. Tap the Add to 'Homescreen' option, then you can notice an icon of 'CTCAE 5.0' added to your devices homescreen instantly.
  4. Tap the icon from homescreen, then 'CTCAE 5.0' will be loaded.
MacOS - Chrome
  1. Go to the website https://oncohub.github.io/pro/.
  2. Reload the page.
  3. Tap the Options button and select the Install CTCAE 5.0.
  4. Then, you can see a new app in the 'user_name/Application/Chrome Apps' folder.
MacOS - Safari
  1. Go to the website https://oncohub.github.io/pro/.
  2. Add https://oncohub.github.io/pro/ to bookmark. [NOTE] Please make sure the URL (not containing 'index.html', '#/tab/home' or other strings).
  3. Launch from the bookmark.
Windows
  1. Go to the website https://oncohub.github.io/pro/.
  2. Tap the more options button.
  3. Tap the Pin to Start option, then you can notice an icon of 'CTCAE 5.0' added to your Start Menu instantly.
  4. Tap the icon from the Start Menu, then 'CTCAE 5.0' will be loaded.

Uninstall

iOS (iPhone)
  1. Go to the Settings and tap on Safari.
  2. Tap the Remove All Website Data.
Android
  1. Open Settings and select Storage.
  2. Tap the Other Apps entry.
  3. Choose an app.
  4. On the app’s info page, tap the Clear Cache option.

Romove 'CTCAE 5.0' from your homescreen and clear the cache of this app out manually. [NOTE] Some action may affect other cached data and files.

MacOS - Chrome
  1. Tap the Options button and select the Install CTCAE 5.0.
MacOS - Safari
  1. In the Safari menu select Preferences.
  2. In the Advanced section, toggle the Show Develop menu to activate.
  3. Click on the Develop in the menu bar and select the Empty Caches.
  4. Close the Safari browser.
Windows
  1. Click the icon with three horizontal dots in the top right of the window to open the More menu.
  2. Click the button that says “Chose what to clear”.
  3. Toggle the check boxes to delete.
  4. Then clear.

You can also unregister the Service Worker and remove cache.
